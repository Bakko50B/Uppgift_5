(async) getInfo() → {void}

Hämta information om kurser och program från en API-endpoint och bearbeta data. Funktionen hämtar data om kurser och program från en specifik API-endpoint, filtrerar och sorterar data, och sparar de mest sökta kurserna och programmen i globala variabler. Funktionen skapar också stapel- och pajdiagram baserat på de bearbetade uppgifterna.

(async) searchLocation() → {void}

Sök plats med Nominatim OpenStreetMap API och flytta kartan samt lägg till markör. Funktionen skickar en sökförfrågan till Nominatim OpenStreetMap API baserat på användarens inmatning, och om en plats hittas flyttas kartan till platsens koordinater och en markör läggs till. Om ingen plats hittas visas ett varningsmeddelande till användaren.
